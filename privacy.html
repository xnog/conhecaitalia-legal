Política de Privacidade

Aplicativo: Conheça Itália Publisher. Última atualização: 28 de junho de 2026.

Esta política descreve como o aplicativo Conheça Itália Publisher ("o aplicativo") trata os dados ao integrar-se com a API do TikTok. O aplicativo é de uso pessoal e interno, operado exclusivamente pelo titular da conta TikTok @conhecaitalia para publicar seus próprios vídeos.

1. Quais dados são acessados

Mediante autorização explícita via TikTok Login, o aplicativo acessa apenas:

  • Informações básicas do perfil do próprio titular (nome de exibição e avatar), pelo escopo user.info.basic, usadas somente para confirmar a conta conectada.
  • Permissão para enviar e publicar vídeos na conta do próprio titular, pelos recursos da Content Posting API (escopo video.upload e Direct Post).

2. Como os dados são usados

Os dados e permissões são utilizados unicamente para fazer upload e publicar vídeos produzidos pelo titular em sua própria conta TikTok. O aplicativo não processa, analisa nem enriquece dados pessoais para qualquer outra finalidade.

3. Dados de terceiros

O aplicativo não coleta, armazena nem compartilha dados de outros usuários do TikTok ou de terceiros. Ele atua exclusivamente sobre a conta do próprio titular que concedeu a autorização.

4. Armazenamento e retenção

Os tokens de acesso emitidos pelo TikTok são mantidos localmente, de forma segura, apenas pelo tempo necessário para realizar as publicações, e podem ser revogados a qualquer momento pelo titular nas configurações de sua conta TikTok. O aplicativo não mantém banco de dados de usuários nem servidores que armazenem conteúdo de terceiros.

5. Compartilhamento

Nenhum dado é vendido, alugado ou compartilhado com terceiros. A única troca de dados ocorre diretamente entre o aplicativo e as APIs oficiais do TikTok.

6. Direitos do titular

O titular pode, a qualquer momento, revogar o acesso do aplicativo em tiktok.com/setting, o que encerra imediatamente qualquer permissão concedida.

7. Contato

Dúvidas sobre esta política: adm@brasilita.com.
